# spam-email-classifier
Spam Email Classifier using Machine Learning and Streamlit.  This project detects whether an email is spam or not using TF-IDF and Naive Bayes algorithm with an interactive web interface.
# 📧 Spam Email Classifier

This project is a Machine Learning based Spam Email Detector built using Python and Streamlit.

## 🚀 Features
- Detects Spam / Not Spam emails
- Uses TF-IDF for text processing
- Naive Bayes algorithm for classification
- Interactive web app using Streamlit

## 🛠️ Technologies Used
- Python
- pandas
- scikit-learn
- Streamlit

## ▶️ How to Run

```bash
streamlit run spam_classifier.py
